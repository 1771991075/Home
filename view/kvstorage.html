Welcome to ConchBrain KV Storage

A KV Storage base on serverless

Set Data

Api: https://storage.conchbrain.workers.dev/bucket/set

Method: POST

Content-Type: application/json

Para: Request body with key and value ( key must be a string )

Example

curl -X POST 'https://storage.conchbrain.workers.dev/bucket/set' \
     -H 'Content-Type: application/json;' \
     -d '{ "key": "hello", "value": "Welcome" }'

Response: "Put successful"

Get Data

Api: https://storage.conchbrain.workers.dev/bucket/get

Method: GET

Para: key

Example

curl https://storage.conchbrain.workers.dev/bucket/get?hello

Response: "Welcome"

Get All Data

Api: https://storage.conchbrain.workers.dev/bucket

Method: GET

Example

curl https://storage.conchbrain.workers.dev/bucket

Response: JsonStr

Delete Data

Api: https://storage.conchbrain.workers.dev/bucket/delete

Method: GET

Para: key

Example

curl https://storage.conchbrain.workers.dev/bucket/delete?hello

Response: "Delete successful"
