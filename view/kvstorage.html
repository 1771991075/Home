Welcome to ConchBrain KV Storage

Set Data

Api: https://storage.conchbrain.club/bucket/set

Method: POST

Content-Type: application/json

Para: Request body with key and value ( key must be a string )

Example

curl -X POST 'https://storage.conchbrain.club/bucket/set' \
-H 'Content-Type: application/json;' \
-d '{ "key": "hello", "value": "Welcome" }'

Response: "Put successful"

Get Data

Api: https://storage.conchbrain.club/bucket/get

Method: GET

Para: key

Example

curl https://storage.conchbrain.club/bucket/get?hello

Response: "Welcome"
