Welcome to ConchBrain KV Storage

WebApi Based Key Value Pair Storage

Set Data

Api: https://storage.conchbrain.workers.dev/demobucket/set

Method: PUT

Content-Type: application/json

Para:
     Custom bucket name (eg. demobucket)
     Request body with key and value ( key must be a string )

Example:
    curl -X PUT 'https://storage.conchbrain.workers.dev/demobucket/set' \
        -H 'Content-Type: application/json;' \
        -d '{ "key": "hello", "value": "Welcome" }'

Response: "Put successful"

Get Data

Api: https://storage.conchbrain.workers.dev/demobucket/get

Method: GET

Para:
     Custom bucket name (eg. demobucket)
     key

Example:
        curl https://storage.conchbrain.workers.dev/demobucket/get?hello

Response: "Welcome"
